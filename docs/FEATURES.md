# Features Overview

## Member Management
- Profile creation and onboarding
- Membership status tracking
- Check-in/check-out system
- Attendance analytics

## Class Scheduling
- Recurring class creation
- Instructor assignment
- Room booking
- Waitlist management

## Payments
- Recurring billing
- Invoice generation
- Payment failure handling
- Revenue reporting

## Mobile App
- iOS and Android support
- Push notifications
- Offline mode
- QR code check-in
