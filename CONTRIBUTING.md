# Contributing to Gym SaaS Platform

Thank you for your interest in contributing! 🎉

## How to Contribute

### 1. Fork & Clone
```bash
git clone https://github.com/Valentinasmith722/gym-saas-platform.git
cd gym-saas-platform
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Create a Branch
```bash
git checkout -b feature/your-feature-name
```

### 4. Make Changes & Test
```bash
npm test
npm run lint
```

### 5. Submit PR
- Describe what you changed and why
- Reference any related issues
- Ensure tests pass

## Areas Where We Need Help

### 🔥 High Priority
- **iOS App (SwiftUI)** — Class booking, member profiles, progress tracking
- **Payment Integration** — Stripe/Solana Pay for subscriptions
- **Push Notifications** — Firebase Cloud Messaging

### 🛠️ Medium Priority
- **Android App (React Native)** — Parity with iOS features
- **Admin Dashboard** — Analytics, member management
- **API Documentation** — OpenAPI/Swagger specs

### 📝 Low Priority
- **UI/UX Improvements** — Design system, accessibility
- **Testing** — Unit tests, E2E tests
- **Localization** — Spanish, Portuguese translations
